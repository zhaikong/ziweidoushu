# 紫微斗数命盘分析系统 - 快速开始指南

## 🚀 立即开始

### 第一步：配置Gemini API密钥

1. 打开项目根目录下的 `.env` 文件
2. 将你的Gemini API密钥填入：

```
GEMINI_API_KEY=你的API密钥
PORT=3000
```

### 第二步：启动服务器

在终端中运行：

```bash
cd /Users/jishudashen/Desktop/sm大师/backend
npm start
```

看到以下信息表示启动成功：

```
🌟 紫微斗数分析系统已启动
📡 服务器运行在: http://localhost:3000
🔑 Gemini API: 已配置
```

### 第三步：打开浏览器

访问：`http://localhost:3000`
